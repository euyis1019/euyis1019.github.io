## 🚀 创建步骤（详细图文教程）

### 第一步：创建新网站

1. 访问 https://sites.google.com/
2. 点击右下角 **"+ 创建"** 按钮
3. 选择 **"空白"** 模板
4. 在顶部输入网站名称，例如 "Your Name"

### 第二步：设置页面布局

**基本设置**:
1. 点击右侧 **"页面"** 标签
2. 重命名主页为 "Home" 或保持默认
3. 添加新页面（可选）:
   - "About"（如果首页内容太多）
   - "News"（新闻/博客）
   - "CV"（可以嵌入 PDF）

**主题设置**:
1. 点击右侧 **"主题"** 标签
2. 选择简洁的主题（推荐 "Simple" 或 "Blank"）
3. 自定义颜色:
   - 主色调: `#1a73e8`（Google 蓝）
   - 强调色: `#202124`（深灰）

### 第三步：创建导航栏

1. 网站自动生成顶部导航
2. 点击导航栏可以:
   - 上传 Logo（推荐使用头像，圆形裁剪）
   - 编辑网站标题
   - 调整导航链接顺序

**Logo 设置建议**:
- 尺寸: 40x40px 到 80x80px
- 格式: PNG（透明背景）
- 形状: 圆形或方形

### 第四步：添加 Hero 横幅

这是最吸引眼球的部分！

1. 点击右侧 **"插入"** 标签
2. 选择 **"图片"** → **"全宽度图片"**
3. 上传你的背景图（推荐至少 1920x600px）

**添加文字到图片上**:
1. 选中图片后，点击工具栏的 **"添加文字"**
2. 输入你的 Slogan，例如: "Come and Build a Silicon Dream with Me"
3. 调整文字样式:
   - 字体大小: 36-48
   - 颜色: 白色
   - 位置: 居中
   - 添加深色背景遮罩（如果文字不够清晰）

**添加社交媒体图标**:
1. 在图片下方添加一个 **"按钮"** 块
2. 设置为图标样式
3. 添加链接到你的社交媒体
4. 常用图标:
   - LinkedIn
   - GitHub
   - Email
   - Google Scholar

### 第五步：添加 About Me 部分

1. 插入 **"文本框"**
2. 输入标题 "About Me"（使用 H2 标题格式）
3. 粘贴你的个人简介
4. 格式化建议:
   - 使用 emoji 增加趣味性（👋 🦾）
   - 分段落，每段 2-3 句话
   - 加粗关键词（公司名、技术名词）

**文本格式技巧**:
```
👋 Hi, I'm [Your Name] ([Chinese Name]), currently serving as...

🦾 In parallel with industry development, I am progressively...
```

### 第六步：创建 Education 卡片

这是展示学历的核心部分！

**方法一：使用图片网格**
1. 插入 **"图片轮播"** 或 **"图片网格"**
2. 设置为 2 列（如果有 2 个学位）
3. 上传学校 Logo
4. 每个图片下方添加文字说明

**方法二：使用卡片布局**（推荐）
1. 插入 **"布局"** → 选择 **"2 列"** 布局
2. 每列包含:
   - 学校 Logo（居中）
   - 学校名称（加粗）
   - 学位信息
   - 时间范围

**示例结构**:
```
┌─────────────────┐  ┌─────────────────┐
│   [School Logo] │  │   [School Logo] │
│   School Name   │  │   School Name   │
│   MS in CS      │  │   PhD in CS     │
│   2022 - 2023   │  │   2026 - Present│
└─────────────────┘  └─────────────────┘
```

### 第七步：添加 Research Interest

1. 插入 **"文本框"**
2. 标题: "Research Interest"（H2）
3. 使用项目符号列表或网格布局
4. 每个研究方向一行

**美化技巧**:
- 使用 **"可折叠组"** 让内容更紧凑
- 添加背景色区分不同部分
- 使用图标或 emoji 标记

### 第八步：创建 Experience 时间线

这是最重要也是最复杂的部分！

**为每个工作经历创建一个块**:

1. 插入 **"布局"** → 选择 **"左图右文"** 或 **"分隔线"**
2. 左侧: 插入公司 Logo（100x100px）
3. 右侧: 
   - 职位标题（H3，加粗）
   - 公司名称（蓝色链接）
   - 时间和地点（灰色文字）
   - 项目描述（项目符号列表）

**完整示例**:
```
┌─────────────┬────────────────────────────────────┐
│             │  VP of Research, Tech Lead         │
│  [Company]  │  Company Name, Singapore           │
│   Logo      │  Sep 2025 – Present                │
│             │                                    │
│             │  Key Projects:                     │
│             │  • Project description 1...        │
│             │  • Project description 2...        │
└─────────────┴────────────────────────────────────┘
```

**项目描述技巧**:
- 使用数据（提升 32%、处理 1.4 trillion tokens）
- 突出技术栈（CUDA、PyTorch、TensorFlow）
- 量化成果（发表论文数、影响范围）

### 第九步：添加页脚

1. 滚动到页面底部
2. 插入 **"文本框"**
3. 添加版权信息和联系方式
4. 设置为灰色背景以区分主内容

**示例文本**:
```
© 2025 Your Name. All rights reserved.
Built with passion for AI and machine learning.
```

## 🎨 高级美化技巧

### 1. 使用一致的配色方案

**推荐配色**（Google Material Design）:
- 主色: `#1a73e8` (蓝色)
- 文字: `#202124` (深灰)
- 次要文字: `#5f6368` (中灰)
- 背景: `#f8f9fa` (浅灰)

**应用方式**:
1. 在 **"主题"** 中设置品牌颜色
2. 链接自动使用主色
3. 标题使用深灰色

### 2. 优化图片

**压缩图片** (加快加载速度):
- 使用 TinyPNG: https://tinypng.com/
- 或 Squoosh: https://squoosh.app/

**Logo 处理**:
- 移除背景: https://remove.bg/
- 统一尺寸: 100x100px
- 格式: PNG

### 3. 添加交互元素

**可折叠部分**:
1. 选择内容块
2. 点击 **"更多选项"** → **"折叠"**
3. 适合长项目描述

**悬停效果**:
- Google Sites 自动添加
- 卡片会在悬停时提升
- 链接会改变颜色

### 4. 嵌入外部内容

**PDF (CV)**:
1. 插入 **"嵌入"** 块
2. 上传 PDF 到 Google Drive
3. 获取共享链接
4. 粘贴到嵌入块

**Google Scholar**:
1. 插入 **"嵌入"** 块
2. 输入你的 Google Scholar URL
3. 自动显示论文列表

## 📱 移动端优化

Google Sites 自动响应式，但你可以：

1. 点击顶部 **"预览"** 图标
2. 选择 **"移动设备"** 视图
3. 检查每个部分的显示效果
4. 调整:
   - 文字大小（移动端建议稍小）
   - 图片尺寸（自动缩放）
   - 布局（自动堆叠）

## 🌐 发布和绑定域名

### 发布网站

1. 点击右上角 **"发布"** 按钮
2. 选择发布对象:
   - **公开**: 任何人都可以访问（推荐）
   - **受限**: 仅限特定人员
3. 获得 Google Sites 免费网址:
   - `https://sites.google.com/view/yourname`

### 绑定自定义域名

**购买域名**（推荐注册商）:
- Namecheap: https://www.namecheap.com/
- Google Domains: https://domains.google/
- GoDaddy: https://www.godaddy.com/

**绑定步骤**:
1. 在 Google Sites 点击 **"设置"** → **"自定义网址"**
2. 输入你的域名（如 `www.yourname.com`）
3. 按照提示在域名注册商添加 DNS 记录:
   ```
   类型: CNAME
   名称: www
   值: ghs.googlehosted.com
   ```
4. 等待 DNS 传播（最多 48 小时）

## 🔧 常见问题

### Q: 如何让 Logo 在图片上居中？
A: 上传图片后，点击图片 → 选择 "全宽度" → 添加文字 → 选择居中对齐

### Q: 可以更改字体吗？
A: 在 "主题" 中可以选择预设字体，但不能上传自定义字体

### Q: 如何添加更多页面？
A: 点击右侧 "页面" 标签 → "+" 按钮 → 输入页面名称

### Q: 图片太大加载慢怎么办？
A: 使用 TinyPNG 压缩图片，或调整图片尺寸到合适大小

### Q: 可以添加自定义 CSS 吗？
A: 不可以，但可以通过嵌入 HTML 块实现部分自定义

### Q: 如何备份网站？
A: Google Sites 自动保存，可以查看版本历史并恢复
