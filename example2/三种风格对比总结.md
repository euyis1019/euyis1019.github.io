## 🎨 风格详解

### 1. Lilian Weng 风格 - 极简学术派

**网址**: https://lilianweng.github.io/

**特点**:
- 纯文字为主，几乎没有图片
- 使用 Jekyll 默认主题
- Markdown 驱动的内容
- 专注于长文章/博客

**优势** ✅:
- 完全免费
- 加载速度极快
- 内容为王，零干扰
- GitHub Pages 托管稳定
- 学术界广泛认可

**劣势** ❌:
- 视觉吸引力低
- 需要 Git 和 Markdown 知识
- 设计过于朴素
- 缺少个性化

**适合你如果**:
- ✅ 会写代码，熟悉 Git
- ✅ 主要发布技术博客
- ✅ 喜欢极简风格
- ✅ 重视加载速度
- ❌ 想要视觉吸引力

**技术栈**:
```
Jekyll + GitHub Pages + Markdown
```

---

### 2. Jason Wei 风格 - 精致极简派

**网址**: https://www.jasonwei.net/

**特点**:
- 现代极简设计
- 有头像和社交链接
- 清晰的视觉层次
- 专业的排版

**优势** ✅:
- 视觉美观且专业
- Squarespace 易于使用
- 自动响应式设计
- 无需维护
- 设计现代时尚

**劣势** ❌:
- 需要付费订阅
- 定制化受限于模板
- 不能完全控制代码
- 迁移困难

**适合你如果**:
- ✅ 愿意付费获得品质
- ✅ 不想写代码
- ✅ 需要专业外观
- ✅ 时间宝贵
- ❌ 预算有限

**技术栈**:
```
Squarespace (托管平台)
```

**费用**:
- Personal Plan: $16/月
- Business Plan: $23/月

---

### 3. Yingchao Li 风格 - 视觉丰富派

**网址**: https://www.yingcli.com/

**特点**:
- 大量图片和 Logo
- 卡片式设计
- Hero 横幅吸睛
- 适合展示丰富经历

**优势** ✅:
- 完全免费
- 最易上手（拖拽式）
- 视觉冲击力强
- Google 托管可靠
- 适合展示职业经历
- 自动移动端优化

**劣势** ❌:
- 定制化最受限
- 必须使用 Google 生态
- SEO 不如静态网站
- 设计选择有限

**适合你如果**:
- ✅ 完全不想写代码
- ✅ 需要快速上线
- ✅ 有丰富工作经历要展示
- ✅ 喜欢视觉化内容
- ❌ 需要完全控制

**技术栈**:
```
Google Sites (拖拽式平台)
```
